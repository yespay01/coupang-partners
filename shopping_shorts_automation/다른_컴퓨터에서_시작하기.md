## 📍 시나리오 1: 내가 다른 장소에서 사용 (사무실 ↔ 집)

### 방법: GitHub에서 받기

```bash
# 1. 원하는 위치로 이동 (예: 바탕화면)
cd Desktop

# 2. GitHub에서 프로젝트 복사
git clone https://github.com/yespay01/coupang-partners.git

# 3. 프로젝트 폴더로 이동
cd coupang-partners/shopping_shorts_automation

# 4. 바탕화면 바로가기 만들기
바탕화면_바로가기_만들기.bat 더블클릭

# 5. 이제부터는 바탕화면 아이콘 더블클릭으로 실행!
```

### ⚠️ 중요: API 키 설정

`.env` 파일에 OpenAI API 키를 추가해야 합니다:

```env
OPENAI_API_KEY=sk-proj-여기에_당신의_API키
OPENAI_MODEL=gpt-4o-mini
```

**방법**:
1. `shopping_shorts_automation` 폴더에서 `.env` 파일 열기
2. `OPENAI_API_KEY=` 뒤에 API 키 붙여넣기
3. 저장

## 🔑 API 키 관리 (중요!)

### 각자 API 키 사용하는 경우

**상황**: 팀원들이 각자 자기 OpenAI 계정 사용

**방법**:
1. `.env` 파일은 Git에 올리지 않음 (이미 .gitignore에 포함됨)
2. 각자 `.env.example` 파일 복사해서 `.env` 만들기
3. 각자 자기 API 키 입력

```bash
# 상대방이 할 일
cd shopping_shorts_automation

# .env.example을 .env로 복사
copy .env.example .env

# .env 파일 열어서 API 키 입력
notepad .env
```

### 팀 공용 API 키 사용하는 경우

**방법 1**: 안전하게 전달
- 카톡/이메일로 API 키만 따로 전달
- 상대방이 직접 `.env` 파일에 입력

**방법 2**: 비밀 관리 도구 사용
- 1Password, Bitwarden 등 사용
- 팀 공유 볼트에 API 키 저장

## 🆘 문제 해결

### 문제 1: Python이 없다고 나옴
**해결**: Python 설치
- https://www.python.org/downloads/
- 설치 시 "Add Python to PATH" 체크!

### 문제 2: Git이 없다고 나옴
**해결**:
- Git 설치: https://git-scm.com/download/win
- 또는 ZIP 다운로드로 받기

### 문제 3: API 키 오류
**해결**:
```
1. .env 파일이 있는지 확인
2. OPENAI_API_KEY 값이 올바른지 확인
3. API 키 앞뒤 공백 제거
```

### 문제 4: 실행이 안 됨
**해결**:
```bash
# 수동으로 실행해서 에러 확인
cd shopping_shorts_automation
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app/main.py
```
