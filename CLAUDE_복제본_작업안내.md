# CLAUDE 복제본 작업 안내

이 디렉터리는 **운영중인 `coupang_auto_blog`의 분리 복제본** 입니다.

## 원칙
- 운영 프로젝트와 혼동 금지
- 실험성 변경/대규모 리팩터링/사업화 준비 작업 전용
- 고객 설치형 표준화 작업은 이 복제본에서 우선 진행

## 반영 절차
1. 복제본에서 설계/구현/검토
2. 검증 완료 후 필요한 변경만 원본 프로젝트에 선택 반영
3. 원본 프로젝트에서만 배포
