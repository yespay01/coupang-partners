## 변경 사항 요약

### 1. "시스템 기능" 카드 삭제 ✅

**파일**: `web/components/admin/AutomationOverview.tsx`

- Firebase Console 링크가 포함된 "시스템 기능" 카드 전체 삭제
- 2열 그리드(`grid gap-6 lg:grid-cols-2`) → 단일 `<section>` (타임라인 카드만 남음)
- `systemFeatures` import 제거

**삭제 이유**: Firebase를 더 이상 사용하지 않으므로 Firebase Console 링크 및 관련 기능 목록 불필요

---

### 2. 대시보드 레이아웃 순서 변경 ✅

**파일**: `web/components/AdminDashboardView.tsx`

**Before (변경 전)**:
```
MetricsSection
AutomationOverview (타임라인 + 시스템기능 2열)
ReportsOverview
AdsReportsOverview / EcpmChart
┌─ grid [1.5fr / 1fr] ──────────────┐
│ 후기승인 워크플로   │ EarningsChart │
└────────────────────────────────────┘
로그 섹션
```

**After (변경 후)**:
```
MetricsSection
AutomationOverview (타임라인만, full-width)
ReportsOverview
AdsReportsOverview / EcpmChart
EarningsChart (full-width, 독립 배치)
후기승인 워크플로 (full-width)
로그 섹션
```

**구체적 변경 내용**:
- `EarningsChart`를 리뷰 섹션 바깥/위에 독립 full-width로 이동
- 리뷰 섹션 `grid-cols-[1.5fr_1fr]` 제거 → 단일 `<section>` full-width
- 리뷰 섹션 내부 `{resolvedView === "overview" && <EarningsChart />}` 삭제
- `showMetrics` 조건으로 EarningsChart 표시 (대시보드 overview에서만)
