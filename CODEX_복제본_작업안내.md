# CODEX 복제본 작업 안내

이 폴더는 **운영 프로젝트 복제본(상품화/설치형 준비용)** 입니다.

## 목적
- 설치형/관리형/구독형 준비 작업
- 구조 개편/실험/기획 검증
- 운영 서비스에 바로 반영하면 안 되는 변경 작업

## 주의
- 운영 반영은 원본 프로젝트에서만 수행
- 이 복제본은 별도 로컬 Git 저장소(`productization-work`)로 관리
- `.env` 비밀값은 복사/커밋 금지

## 현재 우선 작업
- 도메인 하드코딩 제거 (`NEXT_PUBLIC_SITE_URL`)
- 사이트 정보/브랜딩 설정 설계 (`settings.system.site`)
- 설치형/관리형 문서 표준화
